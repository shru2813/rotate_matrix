# rotate_matrix
You are given a n x n 2D matrix A representing an image.
Rotate the image by 90 degrees (clockwise).

Input
[
    [1, 2],
    [3, 4]
 ]

Output
[
    [3, 1],
    [4, 2]
 ]
Input
[
    [1, 2, 3],
    [4, 5, 6],
    [7, 8, 9]
 ]

Output
[
    [7, 4, 1],
    [8, 5, 2],
    [9, 6, 3]
 ]
